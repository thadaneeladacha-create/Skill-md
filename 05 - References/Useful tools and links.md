# Useful Tools and Links

## เครื่องมือที่ใช้บ่อย

### สำหรับสร้าง skill

- **skill-creator** — meta-skill official (ดู [[Skill creator notes]])
- **Markdown editor** — VS Code, Obsidian
- **YAML linter** — ตรวจ frontmatter

### สำหรับ test

- `pytest` — สำหรับ assertion ที่เป็น Python
- `eval-viewer` — HTML viewer ของ skill-creator

### สำหรับ deploy / share

- `package_skill.py` — แพ็กเป็น `.skill` file
- Claude.ai, Claude Code, Cowork — environments ที่ใช้ skill ได้

## Reference Skills (เรียนรู้จาก)

- `pdf` — PDF manipulation
- `docx` — Word document
- `xlsx` — Excel
- `pptx` — Presentation
